Course Description

Course Description In this module you will get an introduction to producing a web layout. We will explore various design principles and techniques to create a modern website using current tools. I invite you to challenge yourself, pay attention to the details, and think outside the box.

Course Schedule

Week 1

Week 1 will serve as an introduction to web development. In this week you will outline and describe the type of website you plan to create. You will also use commonly used tools to publish a simple web site to the internet.

Week 2

In week 2 you will learn about design and aesthetics of a web layout. You will learn design principles such as: common layouts, color schemes and responsive design. You will also learn how to use built-in browser tools to fix bugs, view source code and speed up the design process. You will apply these principles to your site.

Week 3

Week 3 is all about web content. You will learn about the different types of content such as: text, images and video. You will learn when and how to use content in a web layout. You will then add content to your website.

Week 4

Finally in week 4 you will learn about search engine optimization (SEO) and accessibility. You will learn why these things are important and how to incorporate them in your site.
